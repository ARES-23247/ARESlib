Framework Features

Engineered for the most demanding FTC competition environments.

⚡

Zero-Allocation

The 20ms control loop is guaranteed to never trigger a Garbage Collector pause. All math objects are pre-allocated and pooled.

🌍

High-Fidelity Sim

Built-in dyn4j physics engine allows you to test autonomous routines, mechanism collisions, and sensor fusion without a robot.

📊

AdvantageKit-Style Logging

Deterministic log replay and tele-op recording allow you to debug technical failures with frame-by-frame precision.

🛡️

Fault Management

Continuous hardware health monitoring and automated subsystem fallback ensures the robot keeps moving despite sensor failure.

🔄

State Machines

Manage complex mechanism interactions with a robust, enum-driven state machine framework that prevents illegal configurations.

🧪

Headless Testing

Verify your algorithms with pure JUnit 5 tests. Run thousands of simulation cycles in seconds on your dev machine.
