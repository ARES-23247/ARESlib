# ARESLib Controller Mappings

| Device | Input | Action |
| :--- | :--- | :--- |
| driver | <kbd>X Button</kbd> | SPEAKER Mode |
| driver | <kbd>Y Button</kbd> | HP (Human Player) Mode |
| operator | <kbd>X Button</kbd> | POOP (Ground Intake) Mode |
| operator | <kbd>Y Button</kbd> | CLIMB Mode |
| driver | <kbd>A Button (Hold)</kbd> | Align to Tag |
| driver | <kbd>B Button</kbd> | Reset Field-Centric Yaw |
| operator | <kbd>D-Pad Up</kbd> | Elevator: HIGH |
| operator | <kbd>D-Pad Down</kbd> | Elevator: LOW |
| driver | <kbd>Right Bumper</kbd> | Shoot On The Move |
